## 概要

bosim ライブラリは以下のモジュールとファイルから構成される。
各モジュールについてのドキュメントは、対応するディレクトリに含まれる `README.md` に配置している。

*   `base/`
    *   全てのモジュールで広く使用される、基本的なクラスと関数を実装
*   `gpu/`
    *   gpu 関連のモジュールを実装
*   `python/`
    *   Python でフィードフォワード関数を入力するツールを提供
*   `simulate/`
    *   各種シミュレーションのエントリーポイントを実装
*   `circuit.h`
    *   量子回路のクラスとフィードフォワードに関する情報のクラスを実装
*   `operation.h`
    *   操作クラスを実装
*   `result.h`
    *   回路の実行結果のクラスを実装
*   `simulator.h`
    *   状態を回路に通して結果を保持するシミュレータクラスを実装
*   `state.h`
    *   状態クラスを実装
